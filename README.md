# Matrix Operations

This repository contains a small C program that works with two 3x3 matrices and prints:

- the sum of Matrix A and Matrix B
- the product of Matrix A and Matrix B
- the transpose of Matrix A

## Build

Use any C compiler that supports standard C:

```bash
gcc MATRIX_OPS.C -o matrix_ops.exe
```

## Run

```bash
matrix_ops.exe
```

The program will prompt for the 3x3 values for Matrix A and Matrix B, then display the results.
